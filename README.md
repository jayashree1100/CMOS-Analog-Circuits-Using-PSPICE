# Simulation of CMOS Analog Circuits Using PSPICE

## Project Objective
- Develop a **CMOS analog voltage-to-current (V–I) converter** and evaluate its behavior using **PSPICE**.
- **Analyze** the **linearity and performance** of current-mode circuits under different input conditions.
- **Validate** the circuit’s functionality through **simulation waveforms** and **transfer characteristic analysis**.

## Tools & Technologies
- **Software:** ORCAD PSPICE
- **Components:** CMOS transistors, operational amplifier, resistors, voltage sources
- **Logic / Functionality:** Voltage-to-current conversion, negative feedback for precise current control
- **Other:** PSPICE probes for waveform analysis
